Return Process

To return a product, write an email to admin@mitrendi.com containing the below listed:
  • Your full name
  • Your location
  • The order id (This can be found on your Mitrendi profile page, it can be copied and pasted with just a few clicks)
  • A reason for wanting to return the package


  • One last thing, you dont have to be too formal ;-)
We will get back to you as soon as possible.

Please read our return policy to make sure your return meets all the requirements.
